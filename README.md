# Spotify data

## setup env

#### If you don't have uv installed already

```bash
curl -Ls https://astral.sh/uv/install.sh | sh
```

#### else
```bash
uv venv
source .venv/bin/activate 
```

```bash
uv sync
```

## start flask
```bash
python3 app.py
```

## start streamlit
```bash
streamlit run spotify_webpage.py
```
